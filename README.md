# 🛒 Einkaufszettel

Eine moderne Web-App zum Verwalten von Einkaufslisten mit Kategorien.

## 📋 Features

- **Kategorien erstellen** – Organisiere deine Einkaufszettel nach Kategorien (z.B. Obst, Gemüse, Haushalt)
- **Gegenstände verwalten** – Füge Artikel mit Mengen hinzu
- **Abhaken & Zählen** – Markiere gekaufte Gegenstände mit Häkchen
- **Einkaufswagen-Statistik** – Überblick über gekaufte Mengen (zeigt nur abhakten Artikel an)
- **Kategorien sortieren** – Verschiebe Kategorien per Drag & Drop
- **Daten kopieren** – Exportiere deine komplette Liste oder einzelne Kategorien
- **Lokale Speicherung** – Alle Daten werden automatisch gespeichert
- **Responsive Design** – Läuft auf Smartphone, Tablet und Desktop

## 🎯 Bedienung

### Kategorien verwalten

1. **Neue Kategorie erstellen:**
   - Gib einen Namen im Feld "Neue Kategorie..." ein
   - Klicke "+ Kategorie" oder drücke Enter
   - Die neue Kategorie wird automatisch ausgewählt

2. **Zwischen Kategorien wechseln:**
   - Klicke auf einen der Tabs oben
   - Der aktuelle Tab ist farbig hervorgehoben

3. **Kategorien sortieren:**
   - Klicke und halte einen Kategorie-Tab
   - Ziehe ihn an die gewünschte Position

4. **Kategorie löschen:**
   - Klicke auf das "×" am rechten Ende eines Kategorie-Tabs
   - Bestätige den Löschvorgang

5. **Kategorie kopieren:**
   - Klicke auf das "📄" Symbol am linken Ende eines Kategorie-Tabs
   - Die komplette Kategorie wird in die Zwischenablage kopiert

### Gegenstände verwalten

1. **Neuen Artikel hinzufügen:**
   - Gib einen Artikel-Namen im Feld "Neuer Einkaufszettel-Eintrag..." ein
   - Klicke "+ Item" oder drücke Enter

2. **Menge ändern:**
   - Nutze die **−** und **+** Buttons um die Menge zu erhöhen oder senken
   - Oder klicke direkt auf die Zahl und gib einen Wert ein

3. **Artikel abhaken:**
   - Klicke auf das Checkbox-Symbol links vom Artikel
   - Der Artikel wird grau durchgestrichen
   - Die Mengen der abgehakten Artikel werden im "Einkaufswagen" gezählt

4. **Artikel bearbeiten:**
   - Klicke auf den Artikel-Namen um ihn zu ändern
   - Drücke Enter zum Speichern

5. **Artikel löschen:**
   - Klicke auf das "×" rechts vom Artikel

### Einkaufswagen-Zähler

Das Feld "Im Einkaufswagen" zeigt dir die **Gesamtmenge** aller Gegenstände an, die du abhakst:
- Nur abhakten Artikel werden gezählt
- Die jeweiligen Mengen werden addiert
- Der Zähler wird sofort aktualisiert
- Es werden **alle Kategorien** zusammengerechnet

### Weitere Funktionen

- **Alles Kopieren** – Exportiert alle Kategorien und Gegenstände formatiert
- **Alles löschen** – Leert die komplette Liste (mit Bestätigung)

## 💾 Datenspeicherung

Deine Einkaufslisten werden **automatisch gespeichert**:
- Änderungen werden sofort lokal im Browser gespeichert
- Deine Daten bleiben erhalten, auch wenn du die Seite neu lädst
- Die Daten werden lokal auf deinem Gerät gespeichert (nicht auf einem Server)

## 🚀 Verwendung

1. Die App öffnen
2. Kategorien erstellen (z.B. "Bio-Markt", "Drogerie", "Haushalt")
3. Gegenstände hinzufügen
4. Beim Einkaufen abhaken
5. Mit "Alles Kopieren" in dein Smartphone kopieren (wenn nötig)
